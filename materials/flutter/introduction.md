# Flutter basics

Flutter je knihovna pro programovací jazyk [Dart](https://dart.dev/docs)

Flutter renderuje veškeré widgety (komponenty) do canvasu, takže zanecháva veškeré accesibilty feature daného operačního systému (web, adroid, ios, ...)

Flutter je inspirovaný knihovnou react, tím že každý widget je vlastní funkce
